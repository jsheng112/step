TigerTickets: TigerTickets is a centralized platform that allows Princeton University students and staffs to buy, sell, and exchange tickets. It is a final project for COS333: Advanced Programming Techniques taught by Professor Robert Dondero in Spring 2020.

Fall 2018 and Spring 2019

IgniteSTEM Mobile Application: IgniteSTEM is a student-run, nonprofit organization with the goal of empowering K-12 educators with the skills, resources, and networks they need to implement project based learning in public schools. The IgniteSTEM mobile app is an application built to support IgniteSTEM conferences. Attendees can download the app before or during the conference in order to view the scheduled events, signup for certain events with capacity limits, and view the supporting documents for each workshop.
